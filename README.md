# 5秒アニメ（みんなで）Phase2

このZIPには2つ入っています。

- `index.html` : クライアント（GitHub Pagesで公開する）
- `server/` : サーバー（WebSocket）

## 1) クライアント（GitHub Pages）

1. GitHubのリポジトリの一番上（root）に **index.html** を置く  
2. Settings → Pages → Branch を設定（main / root）  
3. 少し待つ → 公開URLで開く

## 2) サーバー（Render / Railway / どこでもOK）

### ローカルで動かす（PCがある場合）
```bash
cd server
npm i
npm start
```
- `http://localhost:8080/health` が `ok:true` なら成功

### Render/Railway
- `server` フォルダをデプロイ
- PORTは環境変数 `PORT` を使う（自動でOK）

## 3) クライアントでサーバーにつなぐ

1. 公開ページを開く  
2. ロビー → 「通信（サーバー）」に **サーバーURL** を入れる  
   - 例: `https://xxxxx.onrender.com`
3. 「保存」→「つなぐ」  
4. 通信がONになったらOK

## 4) ゲームの動き（仕様）

- **部屋を作る**：お題入力 → ID決定 → **1枚目だけ**描ける → 送ったらロビーへ
- **だれかの続きを描く**：公開の未完成部屋から1つ選ぶ → 空きコマの一番若いもの → 送ったらロビーへ  
  - 予約90秒（のこりが表示される）
- **合言葉で入る**：部屋ID + 合言葉 → **自由に制作**
- **自分の作品**：端末内の履歴から **見るだけ**

## 5) 共有（友達と）
- **公開**：誰かが「だれかの続きを描く」で入ってくる
- **プライベート**：IDと合言葉を友達に送る（合言葉で入る）

---

注意：
- サーバーはメモリ保存なので、再起動すると消えます（Phase3で永続化）。
